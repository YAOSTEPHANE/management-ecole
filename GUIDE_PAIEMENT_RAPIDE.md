# Guide Rapide - Paiement des Frais de Scolarité

## 🚀 Étapes rapides

### Pour les Étudiants
1. **Connexion** → Tableau de bord → **"Paiements"**
2. **Sélectionner** le frais à payer → Cliquer **"Payer maintenant"**
3. **Choisir** la méthode de paiement
4. **Remplir** les informations requises
5. **Confirmer** le paiement

### Pour les Parents
1. **Connexion** → Sélectionner l'enfant → **"Paiements"**
2. **Sélectionner** le frais à payer → Cliquer **"Payer maintenant"**
3. **Choisir** la méthode de paiement
4. **Remplir** les informations requises
5. **Confirmer** le paiement

---

## 📱 Mobile Money - Instructions rapides

### Orange Money
1. Composez `*144#`
2. Sélectionnez "Payer une facture"
3. Entrez le numéro et le montant
4. Confirmez avec votre PIN
5. Entrez le code reçu par SMS

### MTN Mobile Money
1. Composez `*126#`
2. Sélectionnez "Paiement de facture"
3. Entrez le numéro et le montant
4. Confirmez avec votre PIN
5. Entrez le code reçu par SMS

### Moov Money
1. Composez `*155#`
2. Sélectionnez "Paiement"
3. Entrez le numéro et le montant
4. Confirmez avec votre PIN
5. Entrez le code reçu par SMS

---

## ⚠️ Points importants

- ✅ Vérifiez toujours le montant avant de confirmer
- ✅ Gardez une copie de votre reçu
- ✅ Le format du numéro : `+237 6XX XXX XXX`
- ✅ Le code de transaction est valide 24h
- ✅ Contactez le support en cas de problème
